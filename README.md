# DevAcademy: apuntes y ejercicios multi-lenguaje

Repositorio personal para centralizar teoría, ejercicios y exámenes de distintas asignaturas y lenguajes. Aquí irán carpetas por lenguaje (C++, Python, Java, SQL, etc.) y materiales variados: práctica guiada, retos, resúmenes y exámenes resueltos.

Qué puedes encontrar
- Carpeta por lenguaje o curso con teoría, ejercicios y exámenes.
- Ejemplos simples para practicar lógica, estructuras de datos y algoritmos.
- Scripts o programas más completos a medida que avance cada materia.

Estado actual
- Solo hay contenido inicial en C++ dentro de 01-C++/ (ejercicios y exámenes).
- main.cpp contiene un ejercicio de gestión de documentos en cajas; el `main()` está vacío y falta enlazar la lectura de operaciones.

Cómo usar y aportar
- Compila los ejemplos según el lenguaje de cada carpeta (en C++ usa `g++ main.cpp -std=c++17 -O2 -o app`).
- Crea tus propias carpetas por lenguaje si quieres experimentar o añadir soluciones.
- Usa issues o notas personales para listar ideas y mejoras pendientes.
